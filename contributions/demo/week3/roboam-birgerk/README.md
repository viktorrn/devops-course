## Description

This demo begins with a narrative where the first task at a new job is improving an already existing CI/CD pipeline (focus on CD). The focus is on refactoring a complex CD pipeline using CircleCI Orbs to make it more maintainable and scalable. The pipeline is expanded easily by integrating additional orbs, demonstrating both the use of existing orbs and the creation of custom ones, including building on top of existing solutions.


**Relevance**

CircleCI is a very popular tool for your CI/CD pipeline and used in major tech companies such as Meta or Adobe.
Orbs are a powerfull and convenient feature oftenly used to simplify your CI/CD workflow, keeping your pipeline's code clean and easy to follow.
