## Description
Management deployment environments with Terragrunt. Write the infrastructure as code once, and deploy the same infrastructure to different environments.

**Relevance**

DevOps is development and operations combined. The similar the dev and prod environments are, the more issues will be able to be spotted earlier. Infrastructure as Code facilitates deployment environment changes through automation. Traditional IaC tools require each deployment environment to have its own IaC definition. With Terragrunt as an IaC orchestration tool, we can reuse the same IaC in different deployment environments. As a result, the DevOps pipeline will be more reliable and predictable, which leads to faster SDLC.
