

Fear No More, we got Loadbalancing at our Core


A demo showcasing a api endpoint managed with a loadbalancer defined in Terraform. In the demo we will do the following:

- Showcase a IaC configruation written in terraform
- We will then simulate "blastnings test" and show response time results
- We will then change the configuration to better scale to the required size
- And the finally run the more optimized solution and show improved results
- Then we make conclusion


For all infrastructure that faces some sort of users/consumers Loadbalancing is going to be required. We will show this using Terraform, which is a tool for managing IaC and managing and distributing traffic to a service endpoint is a central part of any working server infrastructure.
