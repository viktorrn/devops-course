## Description

The demo is about comparing Docker, Docker Compose with Terraform. The ideas is to compare the three methods by implementing the same simple front-end, backend endpoints. The ILO:s are the following,

1. Explain the differences between imperative Docker, declarative Docker Compose, and Infrastructure-as-Code with Terraform.
2. Have a sense for how each approach handles changes and drift.
3. Run the same app via three tools and compare workflows.

The tutorial will be done in KillerCode and run as a tutorial in 5 steps, intro, docker CLI, docker compose, terraform, wrap-up. 

**Relevance**

Understanding the difference between declarative and imperative is a curcial aspect IaC.
