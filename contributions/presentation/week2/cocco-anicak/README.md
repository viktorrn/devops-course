## Description

We want to present the GitHub Actions Supply Chain Attack that initially infected Coinbase and then propagated through the tj-actions/changed-files repository.
The presentation would highlight how attackers exploited vulnerable GitHub Actions workflows and we'll present technical examples of vulnerable vs. secure GitHub Actions YAML files.
It will discuss how automation testing also needs to cover workflow and pipeline security, not just application code.

**Relevance**

The presentation shows that CI/CD pipelines are themselves code and, therefore, vulnerable and important to test and protect.
It illustrates an actual-world DevOps security failure that affected top organisations and it's related to this weeks topic Testing automation, Continuous Integration, Feature flags.
