## Description

The presentation intends to further elaborate on the topic of feature flags and how they make the CI/CD pipeline more streamlined. Furthermore, how can feature flags be used for experimentation of new releases as well as how developers collaboration efforts are simplified. Finally, we intend to discuss what is and isn't a feature flag and how a large tech company (spotify) implement and use feature flags in practice.

**Relevance**
Week 2 concerns CI and Feature flags. These topics are an integral part of DevOps and we aim to provide a basic understanding of the topics as well as an understanding of how these principles are used in practice in large tech companies to provide a more solid foundation of knowledge.
