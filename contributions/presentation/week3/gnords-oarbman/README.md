## Description

This presentation will cover the DevOps technique dark launching. We intend to start with broad concepts and explain how it allows for safe testing in a production environment without impacting the user experience. Finally, we will discuss some real-world challenges, such as where to fork traffic and dark launching mutating services.

**Relevance**

Dark launching allows for validating the performance and reliability of features under real-world traffic. It supports the DevOps goal of quick continous delivery by reducing deployment risks.
